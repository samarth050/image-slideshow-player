# Image Slideshow Player (Python)

A desktop image slideshow application built with Python and Tkinter.

## Features
- Category-based image slideshow
- Embedded slideshow preview
- Optional fullscreen slideshow
- Password-protected image compression using 7-Zip
- Hide / extract images securely
- Indeterminate progress indicator
- Clean, responsive UI layout

## Requirements
- Python 3.9+
- Windows
- 7-Zip installed

## How to Run
```bash
python slideshow.py
